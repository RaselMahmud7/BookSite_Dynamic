অধ্যায় ৪: কন্ডিশনাল স্টেটমেন্ট (Conditionals: if / elif / else)

কন্ডিশনাল স্টেটমেন্ট মানে—যদি শর্ত সত্য হয় তাহলে এক কাজ, না হলে অন্য কাজ। এইভাবে প্রোগ্রাম “সিদ্ধান্ত” নিতে পারে।

কন্ডিশনাল কেন দরকার?
  • ইউজারের ইনপুট অনুযায়ী ভিন্ন আউটপুট দেখাতে
  • ভ্যালিডেশন করতে (ভুল ইনপুট ধরতে)
  • গ্রেডিং/ডিসকাউন্ট/যোগ্যতা চেক করতে
  • একাধিক সিদ্ধান্ত ধাপে ধাপে নিতে

Truthy / Falsy (খুব দরকারি)

Python-এ সবকিছু True/False হিসেবে ব্যবহার করা যায়। যেমন: 0, "", [], None → সাধারণত False ধরা হয়। আর বাকিগুলো সাধারণত True।

১) if — শুধু শর্ত সত্য হলে চলবে

if শর্ত True হলে তার ভিতরের কোড রান হবে। False হলে স্কিপ করবে।

if_basic.py
Code
age = 20

if age >= 18:
    print("You are an adult")
OUTPUT
You are an adult

২) if / else — সত্য হলে এক কাজ, না হলে অন্য কাজ

শর্ত True হলে if ব্লক, আর False হলে else ব্লক চলবে।

if_else.py
Code
temperature = 28

if temperature > 25:
    print("It's hot today")
else:
    print("Weather is nice")
OUTPUT
It's hot today

৩) if / elif / else — একাধিক শর্ত

যখন অনেকগুলো শর্ত পরীক্ষা করতে হয়, তখন elif ব্যবহার হয়। Python উপর থেকে নিচে শর্ত চেক করে—যেই শর্তটা আগে True হবে, সেটাই রান হবে।

conditionals.py
Code
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print("Grade:", grade)
OUTPUT
Grade: B

কিছু Common Pattern (খুব কাজে লাগে)

✅ Range Check (একটা রেঞ্জের ভিতরে কিনা)

যেমন 18 থেকে 30 এর ভিতরে কিনা: 18 <= age <= 30

range_check.py
Code
age = 22

if 18 <= age <= 30:
    print("Young adult range")
else:
    print("Outside range")
OUTPUT
Young adult range
✅ Validation (ইনপুট ঠিক আছে কিনা)

বয়স negative হলে invalid ধরতে পারো।

validation.py
Code
age = -2

if age < 0:
    print("Invalid age")
else:
    print("Valid age")
OUTPUT
Invalid age

Nested if (একটার ভিতরে আরেকটা if)

যখন একটা শর্ত সত্য হলে আরও একটা শর্ত চেক করতে হয়, তখন nested if কাজে লাগে। তবে বেশি nested করলে কোড জটিল হয়ে যায়, তাই সম্ভব হলে and / or দিয়ে শর্ত combine করো।

nested_if.py
Code
age = 22
has_license = True

if age >= 18:
    print("Adult")
    if has_license:
        print("Can drive")
    else:
        print("No license")
else:
    print("Not adult")
OUTPUT
Adult
Can drive

Bonus: One-line if (Ternary)

ছোট decision হলে এক লাইনে লেখা যায়। এটা optional, কিন্তু শিখলে কোড compact হয়।

ternary.py
Code
age = 16
status = "Adult" if age >= 18 else "Minor"
print(status)
OUTPUT
Minor

Common Mistakes (Beginner Warning)

  • = (assign) আর == (compare) গুলিয়ে ফেলো না
  • Indentation ভুল হলে error আসবে (Python-এ indentation খুব গুরুত্বপূর্ণ)
  • একাধিক শর্তের order ভুল হলে ভুল ফল আসতে পারে (if/elif উপর থেকে নিচে)
  • None চেক করতে x is None ব্যবহার করো
অনুশীলন (Practice)
  1. age ভ্যারিয়েবল দিয়ে 18 এর বেশি/কম হলে আলাদা মেসেজ দেখাও।
  2. একটা score নিয়ে A/B/C/F গ্রেড বের করো (if/elif/else)।
  3. ইউজার থেকে একটা সংখ্যা নিয়ে জোড়/বিজোড় প্রিন্ট করো (if + modulus)।
  4. একটা price যদি 100 এর বেশি হয় 10% discount, না হলে 0% discount দেখাও।
  5. Nested if দিয়ে চেক করো: বয়স >= 18 হলে “Adult”, তারপর লাইসেন্স থাকলে “Can drive”।
  6. Ternary ব্যবহার করে “Pass/Fail” বের করো: score >= 50 হলে Pass।
