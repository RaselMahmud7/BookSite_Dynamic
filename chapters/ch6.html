অধ্যায় ৬: লুপস (Loops) — বারবার কাজ অটো করা

একই কাজ বারবার করতে হলে আমরা Loop ব্যবহার করি। যেমন ১ থেকে ৫ পর্যন্ত প্রিন্ট করা, বা একটা লিস্টের সব আইটেম দেখানো। লুপ মানে—একটা কোড ব্লক বারবার চলবে, যতক্ষণ না নির্দিষ্ট শর্ত শেষ হয়।

Loop কেন দরকার?
  • অনেকগুলো ডেটার উপর একই কাজ করতে (list/array)
  • repetitive কাজকে short এবং clean করতে
  • pattern/series print করতে
  • কোনো শর্ত পূরণ না হওয়া পর্যন্ত কাজ চালাতে

১) for লুপ (range দিয়ে)

for লুপ সাধারণত ব্যবহার হয় যখন আমরা জানি কতবার লুপ চলবে। range(start, stop) এ stop include হয় না। তাই range(1, 6) মানে ১ থেকে ৫।

for_range.py
Code
for i in range(1, 6):
    print(i)
OUTPUT
1
2
3
4
5

২) for লুপ (list এর উপর)

List-এর সব item একটার পর একটা দেখতে সবচেয়ে clean হলো: for item in list।

for_list.py
Code
fruits = ["Apple", "Orange", "Banana"]

for fruit in fruits:
    print("Fruit:", fruit)
OUTPUT
Fruit: Apple
Fruit: Orange
Fruit: Banana

৩) while লুপ (শর্ত যতক্ষণ সত্য)

while লুপ চলবে যতক্ষণ শর্ত True থাকবে। এখানে সবচেয়ে বড় বিপদ হলো infinite loop — মানে লুপ কখনো শেষ হবে না। তাই while-এর ভিতরে এমন কিছু থাকতে হবে যাতে শর্ত একসময় False হয় (যেমন count += 1)।

while_basic.py
Code
count = 0

while count < 3:
    print("count =", count)
    count += 1
OUTPUT
count = 0
count = 1
count = 2

৪) break / continue (Loop control)

continue মানে: এই iteration বাদ দিয়ে পরের iteration এ যাবে। break মানে: লুপ একদম বন্ধ করে বের হয়ে যাবে।

break_continue.py
Code
for n in range(1, 6):
    if n == 3:
        continue
    if n == 5:
        break
    print("n =", n)
OUTPUT
n = 1
n = 2
n = 4

৫) Nested loops (এক লুপের ভিতরে আরেক লুপ)

Grid / table / pattern টাইপ কাজের জন্য nested loop ব্যবহার হয়। বাইরের লুপ একবার চললে, ভেতরের লুপ পুরোটা শেষ হয়ে তারপর বাইরের লুপ পরেরবার চলে।

nested_loops.py
Code
for row in range(2):
    for col in range(3):
        print(f"({row},{col})", end=" ")
    print()
OUTPUT
(0,0) (0,1) (0,2) 
(1,0) (1,1) (1,2)

Common Mistakes (Beginner Warning)

  • range(1, 6) মানে ১-৫, ৬ include হয় না
  • while লুপে update না করলে infinite loop হবে
  • break দিলে লুপ পুরো শেষ হয়ে যায়, continue দিলে শুধু ওই iteration বাদ যায়
  • nested loop বেশি হলে কোড slow/জটিল হতে পারে
অনুশীলন (Practice)
  1. ১ থেকে ২০ পর্যন্ত শুধু জোড় সংখ্যা প্রিন্ট করো (for + if)।
  2. একটা list বানাও এবং সব item প্রিন্ট করো (for list)।
  3. while দিয়ে ৫ থেকে ১ পর্যন্ত কাউন্টডাউন করো।
  4. for লুপে ১ থেকে ১০ পর্যন্ত প্রিন্ট করো, কিন্তু ৬ বাদ দাও (continue)।
  5. ১ থেকে ১০ পর্যন্ত প্রিন্ট করো, কিন্তু ৮ এ পৌঁছালে থামাও (break)।
  6. nested loop দিয়ে 3x3 grid coordinate প্রিন্ট করো।
