অধ্যায় ২: ডেটা টাইপ (Data Types)

কম্পিউটারে আমরা যা কিছু নিয়ে কাজ করি—সংখ্যা, লেখা, সত্য/মিথ্যা—সবই ডেটা। কিন্তু সব ডেটা এক রকম নয়। ডেটা টাইপ কম্পিউটারকে বলে দেয়: “এই মানটা কী ধরনের?” এবং “এটার সাথে কী কাজ করা যাবে?”

ডেটা টাইপ কেন গুরুত্বপূর্ণ?
  • একই অপারেটর (+) ভিন্ন টাইপে ভিন্ন কাজ করতে পারে
  • ভুল টাইপ হলে error আসে (যেমন: string আর int যোগ করা)
  • ডেটা ঠিকমতো কনভার্ট/ফরম্যাট করতে পারলে কোড পরিষ্কার থাকে

একই + কিন্তু ফল আলাদা কেন?

সংখ্যা (int/float) হলে + মানে যোগ। স্ট্রিং (str) হলে + মানে জোড়া লাগানো (concatenate)।

plus_behavior.py
Code
a = 3
b = 4
print("Number add:", a + b)

x = "3"
y = "4"
print("String join:", x + y)
OUTPUT
Number add: 7
String join: 34

টাইপ কীভাবে দেখবো? (type())

Python-এ type() ফাংশন দিয়ে কোনো ভ্যারিয়েবলের ডেটা টাইপ জানা যায়।

type_check.py
Code
name = "Rahim"
age = 30
price = 12.75
is_active = True
nothing = None

print(type(name))
print(type(age))
print(type(price))
print(type(is_active))
print(type(nothing))
OUTPUT
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
<class 'NoneType'>

Python-এর সাধারণ ডেটা টাইপ (Must Know)

1) String (str) — লেখা/টেক্সট
"Hello", 'Bangla', "123"

স্ট্রিং হলো লেখার ডেটা। কোটেশনের ভিতরে লিখতে হয়। স্ট্রিং-এর সাথে সংখ্যা সরাসরি যোগ/বিয়োগ করা যায় না।

string_example.py
Code
first = "Hello"
second = "World"
print(first + " " + second)

# f-string (nice formatting)
name = "Rahim"
print(f"Welcome, {name}!")
OUTPUT
Hello World
Welcome, Rahim!
2) Integer (int) — পূর্ণ সংখ্যা
10, 0, -5

দশমিক ছাড়া সংখ্যা হলো int। Python-এ int সাধারণভাবে বড় হতে পারে (মেমোরি পর্যন্ত)।

int_example.py
Code
age = 30
years = 5
print("After 5 years:", age + years)

# integer division vs normal division
print("7 / 3 =", 7 / 3)
print("7 // 3 =", 7 // 3)
OUTPUT
After 5 years: 35
7 / 3 = 2.3333333333333335
7 // 3 = 2
3) Float (float) — দশমিক সংখ্যা
3.14, 12.75, -0.5

দশমিকসহ সংখ্যা হলো float। float দিয়ে গণনা করলে কখনো কখনো ছোট precision issue দেখা দিতে পারে। এটা normal (floating-point behavior)।

float_example.py
Code
pi = 3.14159
r = 2.5
area = pi * r * r
print("Area:", area)

# rounding
print("Rounded:", round(area, 2))
OUTPUT
Area: 19.6349375
Rounded: 19.63
4) Boolean (bool) — True / False
True, False

বুলিয়ান মানে সত্য বা মিথ্যা। Condition (if/else) এ খুব বেশি ব্যবহার হয়। True/False লিখতে হবে (T/F বড় হাতের)।

bool_example.py
Code
is_raining = True
school_closed = False

if is_raining and not school_closed:
    print("Take an umbrella!")
OUTPUT
Take an umbrella!
Truthy / Falsy (খুব দরকারি)

Python-এ কিছু মানকে “False” হিসেবে ধরা হয়—যেমন 0, "", [], None। বাকিগুলো সাধারণত True।

truthy_falsy.py
Code
items = []
if items:
    print("Has items")
else:
    print("Empty list is falsy")

text = ""
if text:
    print("Has text")
else:
    print("Empty string is falsy")
OUTPUT
Empty list is falsy
Empty string is falsy
5) None (NoneType) — “কোনো মান নেই”
None

None মানে “no value / unknown”। None চেক করার সেরা উপায়: is None এবং is not None।

ডেটা টাইপ কাস্টিং (Type Casting)

কাস্টিং মানে এক ধরনের ডেটাকে আরেক ধরনের ডেটায় বদলানো। সাধারণ কাস্টিং: int(), float(), str(), bool()।

casting_basics.py
Code
age_str = "25"
age_int = int(age_str)
print("Age + 5 =", age_int + 5)

price_str = "12.75"
price_float = float(price_str)
print("Price x 2 =", price_float * 2)

num = 99
num_text = str(num)
print("Number as text:", num_text)

print("bool(0) =", bool(0))
print("bool(1) =", bool(1))
print('bool("") =', bool(""))
OUTPUT
Age + 5 = 30
Price x 2 = 25.5
Number as text: 99
bool(0) = False
bool(1) = True
bool("") = False

মনে রাখো: input() সবসময় String দেয়

ইউজার থেকে ইনপুট নিলে সেটা সবসময় str হয়। সংখ্যা হিসেবে ব্যবহার করতে হলে কাস্ট করতে হবে।

input_casting.py
Code
age = int(input("Enter your age: "))
print("After 1 year:", age + 1)
OUTPUT (Example)
Enter your age: 30
After 1 year: 31

Common Mistakes (Beginner Warning)

  • "30" (string) আর 30 (int) এক জিনিস নয়
  • "5" + 3 করলে error হবে (আগে int("5"))
  • int("12.75") দিলে error (float string → float() ব্যবহার করো)
অনুশীলন (Practice)
  1. name (str), age (int), price (float), is_student (bool) ভ্যারিয়েবল বানাও এবং type() দিয়ে টাইপ প্রিন্ট করো।
  2. "100" কে int বানিয়ে 25 যোগ করে প্রিন্ট করো।
  3. ইউজার থেকে দুটি সংখ্যা ইনপুট নিয়ে যোগফল বের করো (input + int casting)।
  4. bool() দিয়ে পরীক্ষা করো: 0, 1, "", "Hi" — কোনগুলো True/False হয়?
  5. একটা float সংখ্যা নিয়ে round(x, 2) ব্যবহার করে 2 decimal এ দেখাও।
