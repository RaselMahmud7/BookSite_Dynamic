অধ্যায় ৭: ফাংশন (Functions) — একবার লিখে বারবার ব্যবহার

ফাংশন হলো একটা কাজের ছোট প্যাকেট (reusable block)। তুমি একবার ফাংশন লিখবে, তারপর দরকার হলে বারবার কল করবে। এতে কোড হয় ছোট, পরিষ্কার, এবং maintain করা অনেক সহজ।

ফাংশন কেন দরকার?
  • একই কাজ বারবার না লিখে reuse করা
  • কোডকে ছোট ছোট অংশে ভাগ করে readable করা
  • ভুল হলে (bug) fix করা সহজ হয়
  • একটা কাজের জন্য একটা নাম (greet, add, login) রাখা যায়

১) ফাংশন লেখার নিয়ম (Syntax)

Python-এ ফাংশন লিখতে def ব্যবহার হয়। তারপর ফাংশনের নাম, তারপর () এর ভিতরে parameter (যদি থাকে), তারপর :। ফাংশনের ভিতরের কোড অবশ্যই indent করা লাগবে।

মনে রাখো:
  • parameter = ফাংশনের ভিতরে যে নামগুলো লিখো (placeholder)
  • argument = ফাংশন কল করার সময় যে আসল মান পাঠাও

২) Default parameter (না দিলে নিজে থেকে মান নেবে)

কোনো parameter-এ default মান দিলে, কল করার সময় সেই argument না দিলেও ফাংশন কাজ করবে। beginner দের জন্য এটা খুবই কাজে লাগে।

default_param.py
Code
def greet(name="Guest"):
    print("Hello,", name)

greet("Rahim")
greet()
OUTPUT
Hello, Rahim
Hello, Guest

৩) return — ফলাফল ফেরত দেওয়া

ফাংশন যদি শুধু print করে—তা শুধু দেখায়। কিন্তু ফাংশন যদি return করে—তাহলে তুমি সেই ফলাফলকে অন্য কাজে ব্যবহার করতে পারবে। return দেখলেই ফাংশন সঙ্গে সঙ্গে শেষ হয়ে যায়।

return_value.py
Code
def multiply(a, b):
    return a * b

result = multiply(4, 5)
print(result)
OUTPUT
20

৪) একাধিক মান return করা (Multiple return)

Python-এ ফাংশন একাধিক মানও return করতে পারে। এটা আসলে tuple হিসেবে return হয়, কিন্তু beginner হিসেবে তুমি “একাধিক মান” হিসেবেই ধরতে পারো।

multiple_return.py
Code
def calc(a, b):
    s = a + b
    d = a - b
    return s, d

sum_val, diff_val = calc(10, 3)
print("Sum:", sum_val)
print("Diff:", diff_val)
OUTPUT
Sum: 13
Diff: 7

৫) Scope (ভ্যারিয়েবল কোথায় কাজ করবে?)

ফাংশনের ভিতরে তৈরি করা ভ্যারিয়েবলকে বলে local ভ্যারিয়েবল। এটা ফাংশনের বাইরে সাধারণত পাওয়া যায় না। (Scope এর বিস্তারিত তুমি আলাদা Chapter-এ দিবে, এখানে শুধু idea রাখলাম)

local_scope.py
Code
def demo():
    x = 10  # local
    print("Inside:", x)

demo()
OUTPUT
Inside: 10

Common Mistakes (Beginner Warning)

  • def লাইনের শেষে : দিতে ভুল করা
  • function body indent না করা
  • return আর print এক জিনিস ভাবা (return = ফেরত, print = শুধু দেখায়)
  • parameter/argument গুলিয়ে ফেলা
অনুশীলন (Practice)
  1. add(a, b) নামে ফাংশন বানিয়ে যোগফল return করো।
  2. is_even(n) নামে ফাংশন বানিয়ে জোড় হলে True, না হলে False return করো।
  3. full_name(first, last) দিয়ে পূর্ণ নাম বানিয়ে return করো।
  4. একটা default parameter দিয়ে greet ফাংশন বানাও।
  5. একটা ফাংশন বানাও যা ২টা সংখ্যা নিয়ে (sum, diff) দুটোই return করবে।
