অধ্যায় ৩: অপারেটর (Operators)

প্রোগ্রামিংয়ে শুধু ডেটা রাখলেই হয় না — সেই ডেটার উপর কাজও করতে হয়। যেমন যোগ-বিয়োগ, তুলনা, শর্ত চেক, মান আপডেট ইত্যাদি। এই কাজগুলো করানোর জন্য আমরা ব্যবহার করি Operators। অপারেটর হলো কিছু বিশেষ চিহ্ন বা শব্দ, যা Python-কে বলে দেয় “কী কাজ করতে হবে”।

Operator কী?

Operator হলো এমন একটি প্রতীক/কীওয়ার্ড যেটা এক বা একাধিক মানের (operand) উপর একটি অপারেশন চালায়। যেমন 10 + 5 এ + হলো operator, আর 10 এবং 5 হলো operands।

১) গাণিতিক অপারেটর (Arithmetic Operators)

সংখ্যা নিয়ে কাজ করার জন্য গাণিতিক অপারেটর ব্যবহার হয়।

  • + যোগ
  • - বিয়োগ
  • * গুণ
  • / ভাগ (ফল float আসে)
  • // ফ্লোর ডিভিশন (পূর্ণ ভাগফল)
  • % মডুলাস (ভাগশেষ)
  • ** পাওয়ার (সূচক)
arithmetic_ops.py
Code
x = 20
y = 3

print("Add:", x + y)
print("Sub:", x - y)
print("Mul:", x * y)
print("Div:", x / y)
print("Floor:", x // y)
print("Mod:", x % y)
print("Power:", x ** y)
OUTPUT
Add: 23
Sub: 17
Mul: 60
Div: 6.666666666666667
Floor: 6
Mod: 2
Power: 8000
Extra: Modulus (%) কেন দরকার?
  • জোড়/বিজোড় চেক: n % 2 == 0
  • ঘড়ি/চক্রাকারে হিসাব: 13 ঘন্টা পরে সময় বের করা
  • লুপে pattern তৈরি

২) অ্যাসাইনমেন্ট অপারেটর (Assignment Operators)

ভ্যারিয়েবলে মান সেট/আপডেট করতে assignment operators ব্যবহার হয়। = দিয়ে মান বসানো হয়, আর += এর মতো শর্টকাট দিয়ে আপডেট করা হয়।

  • = assign
  • += add & assign
  • -= subtract & assign
  • *= multiply & assign
  • /= divide & assign
  • //= floor divide & assign
  • %= mod & assign
  • **= power & assign
assignment_ops.py
Code
score = 10
score += 5
print("After += 5:", score)

score *= 2
print("After *= 2:", score)

score -= 3
print("After -= 3:", score)
OUTPUT
After += 5: 15
After *= 2: 30
After -= 3: 27

৩) তুলনামূলক অপারেটর (Comparison Operators)

দুটি মান তুলনা করে ফল হিসেবে True/False দেয়। এগুলো বেশি ব্যবহার হয় if শর্তে।

  • == equal
  • != not equal
  • > greater than
  • < less than
  • >= greater or equal
  • <= less or equal
comparison_ops.py
Code
a = 25
b = 18

print("a == b:", a == b)
print("a != b:", a != b)
print("a > b :", a > b)
print("a < b :", a < b)
print("a >= 25:", a >= 25)
print("b <= 18:", b <= 18)
OUTPUT
a == b: False
a != b: True
a > b : True
a < b : False
a >= 25: True
b <= 18: True

৪) লজিক্যাল অপারেটর (Logical Operators)

একাধিক শর্ত একসাথে চেক করতে logical operators ব্যবহার হয়।

  • and → দুটো শর্তই True হলে True
  • or → অন্তত ১টা শর্ত True হলে True
  • not → শর্তের উল্টো মান
logical_ops.py
Code
age = 20
has_id = True

if age >= 18 and has_id:
    print("You can enter!")

is_weekend = False
if is_weekend or age < 12:
    print("Discount available")
else:
    print("No discount")
OUTPUT
You can enter!
No discount

৫) Membership Operators (in / not in)

কোনো তালিকা/স্ট্রিং/সেট-এর ভিতরে কোনো মান আছে কিনা চেক করতে in ব্যবহার হয়। এটা lists/strings এ খুব দরকার।

membership_ops.py
Code
fruits = ["Apple", "Banana", "Mango"]

print("Banana" in fruits)
print("Grape" not in fruits)

text = "Hello World"
print("World" in text)
OUTPUT
True
True
True

৬) Identity Operators (is / is not)

is আর == এক না। == মানে value সমান কিনা, আর is মানে একই object কিনা। Beginner level-এ সবচেয়ে বেশি ব্যবহার হবে None চেক করার জন্য: x is None

identity_ops.py
Code
x = None

if x is None:
    print("x is None")

y = 10
if y is not None:
    print("y has value")
OUTPUT
x is None
y has value

৭) Operator Precedence (কোনটা আগে হবে?)

একই লাইনে অনেক অপারেটর থাকলে কিছু অপারেটর আগে কাজ করে। সহজভাবে: () আগে, তারপর **, তারপর * / // %, তারপর + -, তারপর comparisons, তারপর not → and → or। সন্দেহ হলে bracket ব্যবহার করো।

precedence.py
Code
result1 = 2 + 3 * 4
result2 = (2 + 3) * 4

print("Without ():", result1)
print("With ()   :", result2)
OUTPUT
Without (): 14
With ()   : 20

Common Mistakes (Beginner Warning)

  • = assign, আর == compare — গুলিয়ে ফেলো না
  • / সবসময় float দেয়, পূর্ণ ভাগফল চাইলে //
  • ভাগশেষ চাইলে % ব্যবহার
  • None চেক করতে == None না, is None ব্যবহার করো
অনুশীলন (Practice)
  1. একটা সংখ্যা n নাও। n জোড় নাকি বিজোড় বের করো (modulus)।
  2. a = 10, b = 3 নিয়ে / আর // এর পার্থক্য দেখাও।
  3. score ভ্যারিয়েবলে 50 দিয়ে শুরু করে +=, *= ব্যবহার করে আপডেট করো।
  4. দুটি বয়স নিয়ে তুলনা করো: বড় কে? (comparison operators)
  5. একটা লিস্ট বানাও এবং in দিয়ে চেক করো কোন ফল আছে কিনা।
  6. একটা মান None হলে মেসেজ দেখাও (is None)।
  7. 2 + 3 * 4 আর (2 + 3) * 4 তুলনা করো।
