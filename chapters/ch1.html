অধ্যায় ১: ভ্যারিয়েবল (Variables)

ভ্যারিয়েবলকে তুমি ভাবতে পারো একটা নাম দেওয়া বাক্স (box)। এই বাক্সের ভিতরে তুমি তথ্য রাখবে—যেমন নাম, বয়স, দাম, নম্বর ইত্যাদি। পরে দরকার হলে সেই নাম ধরে তথ্য বের করতে পারবে।

ভ্যারিয়েবল কেন দরকার?
  • তথ্য ধরে রাখা (store)
  • পরে ব্যবহার করা (reuse)
  • মান পরিবর্তন করা (update)
  • হিসাব/লজিক করা (calculate)

`=` মানে কী?

Python-এ `=` মানে “সমান” নয়। এটা মানে assign করা। অর্থাৎ, ডান পাশের মানটা বাম পাশের ভ্যারিয়েবলে রাখা।

ভ্যারিয়েবল নাম দেওয়ার নিয়ম (Naming Rules)

  • নামের ভিতরে letter / number / underscore (_) থাকতে পারে
  • নাম number দিয়ে শুরু করা যাবে না
  • Python case-sensitive → name আর Name আলাদা
  • reserved keyword (if/for/def/while) ভ্যারিয়েবল নাম হতে পারবে না
  • অর্থপূর্ণ নাম ব্যবহার করো (age, total_price)
ভুল বনাম সঠিক (❌ vs ✅)
❌ ভুল
  • 1name (number দিয়ে শুরু)
  • my name (space আছে)
  • if (keyword)
✅ সঠিক
  • name
  • total_price
  • student_age
variables_basic.py
Code
name = "Rahim"
age = 30

print("Name:", name)
print("Age:", age)
OUTPUT
Name: Rahim
Age: 30

মান পরিবর্তন (Re-assign)

ভ্যারিয়েবল যেহেতু “চলক”, তাই এর মান পরে পরিবর্তন করা যায়। নতুন মান assign করলে পুরনো মানটা replace হয়ে যায়।

reassign_points.py
Code
points = 100
print("Initial:", points)

points = 120
print("Updated:", points)

points = points + 50
print("Final:", points)
OUTPUT
Initial: 100
Updated: 120
Final: 170

None কী? (No Value Yet)

None মানে: এই ভ্যারিয়েবলে এখনো কোনো মান সেট করা নেই। এটা 0 বা "" (empty string) না। None হলো “no value / unknown”.

none_check.py
Code
favorite_color = None

if favorite_color is None:
    print("Not set yet")

favorite_color = "Blue"

if favorite_color is not None:
    print("Now:", favorite_color)
OUTPUT
Not set yet
Now: Blue

Extra (যা জানলে কাজ অনেক সহজ হবে)

এগুলো শুধু ভ্যারিয়েবল ব্যবহারকে আরও স্মার্ট করে। ডেটা টাইপের বিস্তারিত আমরা Chapter 2-এ করব।

variables_extra.py
Code
# Multiple assignment
x, y = 5, 10
print("x =", x, "y =", y)

# Swap (Python way)
a = "Apple"
b = "Banana"
a, b = b, a
print("a =", a, "b =", b)

# Update shortcut
points = 100
points += 20
print("points =", points)

# Clean output (f-string)
name = "Rahim"
print(f"Hello, {name}!")
OUTPUT
x = 5 y = 10
a = Banana b = Apple
points = 120
Hello, Rahim!
