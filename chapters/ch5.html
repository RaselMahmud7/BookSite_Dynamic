অধ্যায় ৫: লিস্ট (Python List) — একসাথে অনেক ডেটা

একসাথে অনেকগুলো ডেটা রাখতে চাইলে Python-এ আমরা List ব্যবহার করি। List হলো ordered collection — মানে, ভিতরের আইটেমগুলোর একটা নির্দিষ্ট ক্রম থাকে। List-এর সবচেয়ে ভালো দিক: তুমি চাইলে নতুন আইটেম যোগ করতে পারো, মুছে ফেলতে পারো, পরিবর্তন করতে পারো।

List-এর বৈশিষ্ট্য (Must Know)
  • Ordered → index আছে (0 থেকে শুরু)
  • Mutable → value বদলানো যায়
  • Duplicate থাকতে পারে
  • Mixed types রাখা যায় (int, str, bool এক লিস্টে)

১) List তৈরি + Index দিয়ে দেখা

List বানাতে [ ] ব্যবহার হয়। প্রথম আইটেমের index 0। শেষ আইটেম পেতে -1 ব্যবহার করা যায়।

lists_basic.py
Code
fruits = ["Apple", "Orange", "Banana", "Guava"]

print("All:", fruits)
print("First:", fruits[0])
print("Last:", fruits[-1])
OUTPUT
All: ['Apple', 'Orange', 'Banana', 'Guava']
First: Apple
Last: Guava

২) List-এর দৈর্ঘ্য (len) + Item পরিবর্তন

List-এ কতগুলো item আছে জানতে len() ব্যবহার হয়। আর list mutable, তাই fruits[2] = "Mango" এভাবে value বদলানো যায়।

len_modify.py
Code
fruits = ["Apple", "Orange", "Banana", "Guava"]

print("Length:", len(fruits))

fruits[2] = "Mango"
print("After change:", fruits)
OUTPUT
Length: 4
After change: ['Apple', 'Orange', 'Mango', 'Guava']

৩) List-এ যোগ করা (append / insert / extend)

append() → শেষে যোগ করে, insert(index, item) → নির্দিষ্ট জায়গায় যোগ করে, extend() → আরেকটা list যোগ করে (একসাথে অনেক item)।

add_items.py
Code
fruits = ["Apple", "Orange"]

fruits.append("Banana")
fruits.insert(1, "Pineapple")
fruits.extend(["Guava", "Mango"])

print(fruits)
OUTPUT
['Apple', 'Pineapple', 'Orange', 'Banana', 'Guava', 'Mango']

৪) List থেকে মুছে ফেলা (remove / pop / del / clear)

remove(item) → নির্দিষ্ট value মুছে ফেলে (প্রথম ম্যাচ), pop(index) → index অনুযায়ী মুছে ফেলে এবং value ফেরত দেয়, del → index দিয়ে ডিলিট, clear() → সব মুছে ফেলে।

remove_items.py
Code
fruits = ["Apple", "Orange", "Banana", "Guava", "Banana"]

fruits.remove("Banana")     # removes first Banana
removed = fruits.pop(1)     # removes index 1 and returns it
del fruits[0]               # deletes item at index 0

print("List:", fruits)
print("Removed by pop:", removed)
OUTPUT
List: ['Guava', 'Banana']
Removed by pop: Orange

৫) Slicing (এক টুকরো অংশ বের করা)

list[start:end] দিয়ে slicing করা যায়। এখানে end indexটা include হয় না (exclude)।

slicing.py
Code
nums = [10, 20, 30, 40, 50, 60]

print(nums[1:4])   # 20,30,40
print(nums[:3])    # first 3
print(nums[3:])    # from index 3 to end
print(nums[-3:])   # last 3
OUTPUT
[20, 30, 40]
[10, 20, 30]
[40, 50, 60]
[40, 50, 60]

৬) Useful List Methods (কাজের জিনিস)

  • sort() → ছোট থেকে বড় সাজায় (in-place)
  • reverse() → উল্টে দেয়
  • count(x) → কতবার আছে
  • index(x) → প্রথম কোথায় আছে
  • copy() → shallow copy (beginner এ যথেষ্ট)
list_methods.py
Code
nums = [5, 2, 9, 2, 1]

print("Count 2:", nums.count(2))

nums.sort()
print("Sorted:", nums)

nums.reverse()
print("Reversed:", nums)

print("Index of 9:", nums.index(9))
OUTPUT
Count 2: 2
Sorted: [1, 2, 2, 5, 9]
Reversed: [9, 5, 2, 2, 1]
Index of 9: 0

৭) in / not in (List-এ আছে কিনা)

কোনো item list-এর ভিতরে আছে কিনা চেক করতে in ব্যবহার করো।

membership.py
Code
fruits = ["Apple", "Orange", "Banana"]

print("Banana" in fruits)
print("Mango" not in fruits)
OUTPUT
True
True

Common Mistakes (Beginner Warning)

  • Index 0 থেকে শুরু হয় — প্রথম item list[0]
  • ভুল index দিলে IndexError হবে
  • remove() value দিয়ে কাজ করে, pop() index দিয়ে
  • sort() list-কে বদলে দেয় (in-place), নতুন list রিটার্ন করে না
অনুশীলন (Practice)
  1. ৫টা ফলের নাম দিয়ে একটা list বানাও।
  2. প্রথম ও শেষ item প্রিন্ট করো (index 0 এবং -1)।
  3. একটা item যোগ করো (append) এবং মাঝখানে আরেকটা যোগ করো (insert)।
  4. একটা item মুছে ফেলো (remove) এবং একটা item pop করে বের করো।
  5. একটা নাম list-এর ভিতরে আছে কিনা in দিয়ে চেক করো।
  6. একটা সংখ্যার list নিয়ে sort করে দেখাও।
